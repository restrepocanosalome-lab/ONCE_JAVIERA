📬 Formulario de Contacto - Mejoras y Personalización

✨ Descripción

Este proyecto consiste en un formulario de contacto desarrollado en HTML y CSS.
Se realizaron mejoras tanto en la funcionalidad como en el diseño visual mediante los retos propuestos.

⸻

🏆 Cambios realizados

🔹 Reto 1: Cambio de identidad visual

Se modificaron variables CSS para darle un nuevo estilo al formulario:
	•	🎨 Color principal cambiado a #16f9ee
	•	🌑 Color primario oscuro actualizado a #ea0cd8
	•	🔵 Bordes redondeados aumentados de 8px a 20px

📌 Resultado:
El formulario ahora tiene una apariencia más moderna, llamativa y con colores más vibrantes.

⸻

🔹 Reto 2: Nuevo campo “Edad”

Se agregó un nuevo campo dentro del formulario HTML:
	•	📍 Ubicación: Entre Asunto y Mensaje
	•	🔢 Tipo de dato: number
	•	⚙️ Restricciones:
	•	Mínimo: 15
	•	Máximo: 99
	•	Campo obligatorio (required)

📌 Resultado:
Ahora el formulario también captura la edad del usuario con validación automática.

⸻

🔹 Reto 3: Botón principal mejorado

Se modificó el botón principal (.btn-primario) para hacerlo más atractivo:
	•	💥 Sombra más intensa y profunda
	•	🚀 Efecto hover con:
	•	Aumento de tamaño (scale)
	•	Elevación (translateY)
	•	Sombra exagerada

📌 Resultado:
El botón tiene un efecto visual más dinámico e impactante.
